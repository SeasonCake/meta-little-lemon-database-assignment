# Little Lemon Database Capstone Project

## Project Description
A database management system for Little Lemon restaurant, including a booking system, data analysis and visualization.

## Files
- `LittleLemonDM.mwb` - MySQL Workbench data model
- `LittleLemonDM.png` - ER diagram
- `LittleLemonDB_ER_diagram.sql` - Database schema SQL file
- `stored_procedures.sql` - Stored procedures (GetMaxQuantity, ManageBooking, UpdateBooking, AddBooking, CancelBooking)
- `LittleLemonDB.ipynb` - Python database client
- `LittleLemonDB.twb` - Tableau workbook
- `import_data.py` - Data import script

## Setup
1. Run `LittleLemonDB_ER_diagram.sql` in MySQL Workbench to create the database
2. Run `import_data.py` to import data (update password in the script first)
3. Run `stored_procedures.sql` to create stored procedures
4. Open `LittleLemonDB.ipynb` in Jupyter to interact with the database
